# Halı Mozaik v4.0

İnteraktif Kolektif Dokuma Enstalasyonu

## Özellikler

*   **Host Ekranı:** 3D halı simülasyonu, yün shader efektleri, dinamik ışıklandırma.
*   **Client (Mobil) Ekranı:** Dokuma paneli, simetri modları (4-Ayna, Radyal), jiroskop ile "Savur-Gönder" özelliği.
*   **Teknolojiler:** React, Three.js (R3F), Socket.IO, Node.js.

## Kurulum

1.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```

2.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```

3.  Socket sunucusunu başlatın (ayrı terminalde):
    ```bash
    npm run server
    ```

4.  Tarayıcıda `http://localhost:3002` adresine gidin.
